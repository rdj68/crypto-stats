Welcome to the Crypto Stats API

About

This API provides cryptocurrency statistics and price deviation calculations. Below are the available endpoints:

Endpoints

1. Get Cryptocurrency Stats

Endpoint: /stats
Query Parameter: coin (required, e.g., "bitcoin", "ethereum", "matic-network")
Example: /stats?coin=bitcoin

2. Get Price Deviation

Endpoint: /deviation
Query Parameter: coin (required, e.g., "bitcoin", "ethereum", "matic-network")
Example: /deviation?coin=bitcoin
